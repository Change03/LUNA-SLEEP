# LUNA&SLEEP Mall

루나앤슬립 브랜드 소개서와 허가받은 상품 이미지를 기반으로 만든 자사몰 프로토타입입니다.

## 실행

```bash
npm run dev
```

로컬 주소:

```text
http://localhost:5173
```

## 구성

- 쇼핑 홈
- 상품 상세 페이지
- 장바구니와 주문 mock
- 마이페이지 mock
- 관리자 상품/자산/주문/쿠폰/리뷰/Q&A 화면
- 허가받은 이미지 import 스크립트

## 자산 권한

자산 사용 범위와 출처는 `docs/asset-permission.md`, `assets/permissions.json`, `assets/product-content-sources.json`에 기록되어 있습니다.
